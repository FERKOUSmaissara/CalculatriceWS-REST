# Projet SOAP & REST - Calculatrice

## Structure
- REST : API REST avec Jersey (Hello + Todo)
- SOAP : Service SOAP (calculatrice)
- ClientWS : client SOAP

## Lancer REST
cd REST
mvn clean install
(deployer sur Tomcat)

## Lancer SOAP
cd SOAP
mvn clean install
(deployer sur Tomcat)

## Tester SOAP
Lancer ClientWS
